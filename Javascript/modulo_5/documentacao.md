Estuturas de Dados 
1-Array: são estruturas que são tipadas, porem em javascript
2-Pilha: são estruturas que formam uma coleção ordenada de itens que segue o principio LIFO
(Last In First Out), ou seja, o último a entrar é o primeiro a sair.

obs: Math.floor(): Garante que o resultado seja arredondado para baixo (embora, no caso do % 2, o resultado sempre será 0 ou 1, o Math.floor é uma segurança comum em algoritmos de conversão de base para evitar resíduos de ponto flutuante).

Em JS temos os metodos de arredondamentos:
Math.ceil(x) -> Arredonda para cima
Math.floor(x) -> Arredonda para baixo
Math.round(x) -> Arredonda para o mais próximo

3-Fila: são estruturas que formam uma coleção ordenada de itens que segue o principio FIFO (First In First Out), ou seja, o primeiro a entrar é o primeiro a sair.
